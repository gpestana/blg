blg
===

Blog engine implemented with Python - Flask framework - with Admin Panel and search engine.


Main modules:
  - Flask
  - PostgreSQL
  - Flask­Login
  - Flask­SQLAlchemy
  - WTForms
  - SQLAlchemy
